# medical-docs-de

Textbausteine und Vorlagen zur medizinischen Dokumentation auf Deutsch für espanso.

Deutsche Übersetzung von [medical-docs](https://github.com/wressl/medical-docs) (Bill Ressl), angepasst für deutschsprachige Befunde/Arztbriefe.

## Installation

```
espanso install medical-docs-de --git https://github.com/ArcticSpaceFox/medical-docs-ger --external
```

## Update

```
espanso package update medical-docs-de
```
